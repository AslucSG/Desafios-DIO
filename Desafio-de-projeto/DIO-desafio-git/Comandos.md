# Comandos vistos em aula

## cd
Navega entre as pastas	
### cd ..
Retorna para a pasta anterior

## ls
Lista as pastas/ arquivos
### ls -a
Lista pasta/ arquivos ocultos

## mkdir
Cria pastas/ arquivos

## rm -rf
Apaga pastas/ arquivos

## git init
Cria um repositório

## git add 
Adiciona o arquivo no repositório

## git commit -m
"commita" o arquivo e permite carregar uma mensagem ao commit

## git status
Verifica se existe alguma pendencia no repositório

## git push 
"Empurra" o repositório para o GitHub

## git pull
"Puxa" o repositóto do GitHub para sua máquina

## git clone
"Clona" um repositório do GitHub para sua máquina

## git config --list
Apresenta todas as configurações do Git na sua máquina

## git config --global user.email "seu email"
Adiciona seu endereço de email ao Git

## git config --global --unset user.email "seu email"
Remove seu endereço de email do Git

## git config --global user.name "apelido"
Adiciona seu apelido ao Git

## git config --global --unset user.name "apelido"
Remove seu apelido do Git

## git remote add "apelido" "link do repositório"
Adiciona uma "origin" ao Git
